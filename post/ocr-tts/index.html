Optical Character Recognition is an approach aiming to recognize characters from images (mainly) in this way, thus making it easier to manipulate the text that has been recognized. Text to Speech refers to speech synthesis or an “artificial production of speech”.

Python + PyTesseract + Google Text-to-Speech

PyTesseract - https://pypi.org/project/pytesseract gTTS - https://pypi.org/project/gTTS

Prerequirements ( Ubuntu )

$ sudo apt-get install -y python-gtts
$ sudo apt install tesseract-ocr
$ sudo apt install libtesseract-dev
$ sudo apt-get install tesseract-ocr-eng #english
$ sudo apt-get install tesseract-ocr-por #portugues
$ pip install playsound

Python Code

from gtts import gTTS 
from playsound import playsound 
import pytesseract as ocr

from PIL import Image

text = ocr.image_to_string(Image.open('motivacao.jpg'), lang='por')

var = gTTS(text = text,lang = "pt") 
var.save("text.mp3") 
playsound("text.mp3")
